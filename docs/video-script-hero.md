# Script Vidéo Hero — aiman-renovation.fr

## Concept
Vidéo 8 secondes, caméra fixe avec léger zoom avant/arrière.
Maison alsacienne à colombages. Transformation complète de ruine à rénovée.

## Découpage par timecode

| Seconde | Action vidéo | Texte à afficher (côté) |
|---------|-------------|------------------------|
| 0.0-1.0 | Maison en ruine. Murs fissurés, volets cassés, jardin envahi. Ciel couvert. | GAUCHE: "Votre habitat mérite mieux" |
| 1.0-2.0 | Échafaudage monte. 5 ouvriers en rouge arrivent. Un ouvrier commence à gratter le mur. | DROITE: "Nos artisans prennent le relais" |
| 2.0-3.5 | Ouvriers en action rapide : un sur le toit, un sur la façade, un au jardin. Ancien crépi tombe. | GAUCHE: "19 ans d'expérience" |
| 3.5-5.0 | Isolation posée, enduit appliqué, nouveau toit visible. Couleurs changent. | DROITE: "Chaque détail compte" |
| 5.0-6.5 | Finitions : peinture fraîche, volets neufs installés, jardin débroussaillé. | GAUCHE: "Du sol au plafond" |
| 6.5-8.0 | Résultat final : maison rénovée, lumière chaude, jardin fleuri. Ouvriers fiers. Golden hour. | CENTRE: "Nous rénovons jusqu'au bout de vos rêves" |

## Mapping scroll → vidéo (pour 8s de vidéo, hauteur 500vh)

- 0% scroll → 0.0s (ruine, titre principal)
- 12% → 1.0s (arrivée ouvriers)
- 25% → 2.0s (démolition)
- 43% → 3.5s (construction)
- 62% → 5.0s (finitions)
- 81% → 6.5s (résultat)
- 100% → 8.0s (fin, CTA)
