## 🚨 Erreurs critiques à éviter

1. ❌ **Variations du NAP** entre annuaires → utiliser strict `02-citations-locales.md`
2. ❌ **Achat de backlinks** sur Fiverr / fermes de liens → blacklist Google
3. ❌ **Spammer 30 annuaires en 1 jour** → étaler sur 2 semaines
4. ❌ **Oublier les UTM** sur les liens posts GBP → impossible de tracker
5. ❌ **Sitemap soumis mais pas demandé en indexation** → 30 jours d'attente au lieu de 7
6. ❌ **Une fiche GBP française seulement** → invisible côté Suisse / Allemagne
